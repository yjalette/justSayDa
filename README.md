## Technical
DA was built as two separate parts.

### Front End

* HTML5
* CSS3(SASS)
* JavaScript
* jQuery
* Handlebars

### Back End

* Node.js
* Express.js
* MySql
* Shopify button

### Responsive
* The app is responsive and optimized for both desktop and mobile viewing and use.

### Security
* User passwords are encrypted using bcrypt.js
